# MVP

### Цель

Создать работающий терминал.

### Задачи

- [ ] Создать цикл обработки команд.
- [ ] Реализовать приглашение (`webfs>`).
- [ ] Добавить команды:
    - [ ] help
    - [ ] clear
    - [ ] exit
- [ ] Реализовать простой парсер команд.

## Этап 2. Подключение к сайту

### Цель

Научиться получать страницы.

### Задачи

- [ ] Реализовать команду `open`.
- [ ] Проверить доступность сайта.

## Этап 5. Команда ls

### Цель

Получить первый рабочий файловый менеджер.

### Задачи

После

```bash
open https://example.com
```

команда

```bash
ls
```

должна выводить

```text
about

blog

products

contacts
```

# Результат

После завершения проекта должно получиться полноценное консольное приложение, позволяющее исследовать структуру веб-сайтов так же удобно, как файловую систему Linux.

Проект должен иметь расширяемую архитектуру, позволяющую легко добавлять новые команды, сервисы и анализаторы без изменения существующего кода.
